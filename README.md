# html_homework
Домашняя работа Skypro
Эта работа посвящена изучению видов систем контроля версий.Это поможет мне понять, почему команде разработчиков удобнее хранить и изменять код проекта в СКВ. Чтобы облегчить работу над проектом, в котором участвуют несколько человек, используют системы контроля версий.СКВ позволяют разработчикам создавать, хранить копии одного файла и работать с ними, не затрагивая при этом основной файл.
